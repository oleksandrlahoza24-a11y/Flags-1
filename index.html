📖 ar(> Language Manual

Overview

ar(> (pronounced "Arrow") is a modern scripting language designed to be readable, expressive, and powerful. It compiles to JavaScript and runs in any browser or runtime. Its syntax is designed to be English-like, removing boilerplate while keeping full capability.

Variables

set name = "Alice" -- string set age = 25 -- number set isOn = true -- boolean set items = [1, 2, 3] -- list const PI = 3.14159 -- constant

Functions

fn greet(name) => say "Hello, " + name end fn add(a, b) => return a + b end -- Arrow shorthand fn square(x) -> x * x

Control Flow

-- If / elif / else if age > 18 then say "Adult" elif age == 18 then say "Just 18" else say "Minor" end -- Loops loop 10 times say "Hello!" end for item in items say item end while count < 5 count += 1 end

Classes & Objects

class Player has name, health = 100, x = 0, y = 0 fn init(name) => self.name = name end fn move(dx, dy) => self.x += dx self.y += dy end fn takeDamage(amount) => self.health -= amount if self.health <= 0 then say self.name + " died" end end end set p = new Player("Hero")

Output & Input

say "Hello World" -- console.log say "x = " + x -- string concat ask "Your name?" -> name -- prompt show alert "Warning!" -- alert set val = read input -- input element value

UI Elements

Files & Storage

-- Local storage save "score" = 1000 load "score" -> score -- File operations (with user permission) pick file -> f read file f -> text -- Fetch fetch "https://api.example.com/data" -> result say result.json

Multi-Script (use)

-- In main.ar use "utils.ar" use "player.ar" -- utils.ar exports functions automatically -- Call them directly: set p = new Player("Alice") log(p.name)

Types & Casting

set n = toNum("42") -- 42 set s = toStr(3.14) -- "3.14" set b = toBool(0) -- false type of n -- "number" is n a Number -- true

Error Handling

try riskyFn() catch err say "Error: " + err.message finally cleanup() end

Math & Utilities
